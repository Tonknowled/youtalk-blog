# Блог YouTalk

Проект для портфолио: вёрстка главной страницы блога и страницы статьи для компании YouTalk, которая запускает блог о психотерапии и самопомощи.

## Технологии

- HTML5
- CSS3
- Flexbox / Grid
- БЭМ-методология
- Адаптивная и резиновая вёрстка
- Семантические теги

## Что сделано

- ✨ Две полностью сверстанные страницы: **Блог** (`index.html`) и **Статья** (`article.html`)
- 📱 Адаптив под три основных разрешения: десктоп (3 колонки), планшет (2 колонки), мобильный телефон (1 колонка)
- 🧩 Сетка карточек статей на Grid, шапка и подвал на Flexbox
- 🎨 Псевдоклассы (`:hover`, `:active`) для интерактивности
- ♿️ Базовая доступность (alt для картинок, aria-label для кнопок)
- 🏷 Классы по методологии БЭМ
- 🍔 Бургер-меню для мобильной версии
